# Documentation

This folder contains supporting documentation for the Shopify Customer and Sales Analysis Dashboard.

## File

- `shopify_data_terminology.docx` - Document explaining the meaning of important dataset fields used in the dashboard.

## Purpose

The terminology document explains important Shopify dataset fields including order number, customer ID, invoice date, payment gateway, product type, quantity, subtotal price, total price USD, billing location, and tax details.

It helps users understand the dataset columns before exploring the Power BI dashboard.
